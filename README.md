# Football Board

A recruitment exercise built using Java with OO and TTD approach.

## Running

This project is not an apped meant to be ran as a standalone application. The behaviour of the app can be verified using unit tests.

## Code

The codebase follows following conventions:
- All public non-void methods are assumed to return a non-null value or an `Optional<>`. If the value cannot be returned an `IllegalArgumentException` is to be thrown.
